# Simpons_Character_Classifier

This project creates a model which can detect a few characters from the reputated tv series The Simpons.
The characters include:
  *krusty the clown
  *charles montgomery burns
  *ned flanders
  *homer simpson
  *milhouse van_houten
  *principal skinner
  *chief wiggum
  *moe szyslak
  *lisa simpson
  *marge simpson
  *sideshow bob
  *abraham grampa simpson
  *bart simpson
